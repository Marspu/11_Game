# 🎨 颜色魔法学院

一个专为儿童设计的交互式颜色混合学习应用，通过有趣的方式教授颜色叠加原理。

## 🌟 项目特点

### 科学的教育设计

本项目基于**科学准确**的颜色理论设计，包含两种颜色混合模式：

#### 1. 💡 加色混合（光的混合）
- **原理**：RGB（红绿蓝）三原色光叠加
- **特点**：颜色越加越亮，最终混合成白色
- **应用**：电视、手机、电脑屏幕
- **科学公式**：
  - 红光 + 绿光 = 黄光
  - 绿光 + 蓝光 = 青光
  - 红光 + 蓝光 = 品红光
  - 红光 + 绿光 + 蓝光 = 白光

#### 2. 🎨 减色混合（颜料的混合）
- **原理**：CMY（青品黄）三原色颜料混合
- **特点**：颜色越混越暗，最终混合成深色
- **应用**：打印机、绘画、印刷
- **科学公式**：
  - 青色 + 品红 = 蓝色
  - 品红 + 黄色 = 红色
  - 青色 + 黄色 = 绿色
  - 青色 + 品红 + 黄色 = 黑色

## 🎯 教育目标

1. **理解原色概念**：让孩子理解什么是"原色"，为什么有些颜色不能由其他颜色混合而成
2. **区分两种混合模式**：理解光混合和颜料混合的根本区别
3. **培养科学思维**：通过实验观察→预测结果→验证的学习方法
4. **激发探索兴趣**：游戏化的学习方式，让孩子主动探索

## 🔬 科学原理说明

### 为什么光混合和颜料混合不一样？

**光混合（加色）**：
- 每种颜色的光都在"增加"能量
- 红光 + 绿光刺激我们眼睛的红色和绿色感受器，大脑解读为黄色

**颜料混合（减色）**：
- 颜料通过"吸收"（减去）某些颜色的光来显示颜色
- 青色颜料吸收红光，反射绿光和蓝光
- 品红颜料吸收绿光，反射红光和蓝光
- 混合后，只有蓝光被共同反射，所以我们看到蓝色
